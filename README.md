This is my first **Golang Project**, just started learning it because I'm really curious about what Golang could do.

So basically, I made a todo list app but it is a **CLI Based**. I made it in Git Bash doing all the command line and stuffs, such as
- go mod init tasks
- touch main.go (to make .go file)
- nano main.go (to edit .go file, here you could make your codes)
- go build -o tasks.exe

This todo list app has some features that are:
- **add** : to add your task list
- **list** : to view all the task list you have
- **complete** : to mark a task that has been completed
- **delete** : to delete task
